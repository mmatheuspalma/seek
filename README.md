#### Requirements
Package managers Npm or Yarn ( preference to Yarn, by speed and security )

#### Install

##### Npm
``js
   npm install 
``

##### Yarn
``js
   yarn install 
``

#### Running

##### Npm
``js
   npm start
``

##### Yarn
``js
   yarn start
``

### Changing rules
To change a specific discount rule, you can modify it on the state of main component ( checkout )
